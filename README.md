## Getting Started

### Pre-requisites

- node >= 16
- npm >= 7.20.x

### Local Development

```bash
# Install Dependencies
$ npm i

# Start Apollo Server
$ npm run dev
```

### Unit Tests

```bash
# Run All Unit Tests
$ npm test

# Run Specific Unit Test
$ npm run test:only <path of test>
```

### Generate Schema Types

```bash
$ npm run generate
```

### Linters

```bash
# Lint Source Code
$ npm run lint

# Make the linter fix the Source Code
$ npm run lint:fix
```
